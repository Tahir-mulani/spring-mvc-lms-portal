# 📚 Learning Management System (LMS) Portal

A modern and simple web-based Learning Management System built with Spring MVC, JSP, Maven, and Bootstrap. This project offers an elegant admin experience for managing courses with login-based access and a responsive dashboard.

![Java](https://img.shields.io/badge/Java-17%2B-blue) ![Spring MVC](https://img.shields.io/badge/Spring%20MVC-5.x-brightgreen) ![Maven](https://img.shields.io/badge/Maven-Build-red) ![Tomcat](https://img.shields.io/badge/Tomcat-10.x-yellowgreen)

---

## ✨ Overview

The LMS Portal is designed for educational purposes to demonstrate the fundamentals of Spring MVC architecture, form handling, controller mapping, and JSP-based UI rendering. It provides a clean interface for administrators to manage courses efficiently.

## ▶️ Getting Started

### Prerequisites
Make sure the following are installed:
- Java JDK 17+
- Maven
- Apache Tomcat 10
- Spring Tool Suite / Eclipse

### 1. Clone the Repository
```bash
git clone https://github.com/Tahir-mulani/spring-mvc-lms-portal.git
```

### 2. Import into IDE
1. Open STS or Eclipse
2. Go to File → Import → Existing Maven Projects
3. Select the project folder
4. Click Finish

### 3. Build the Project
```bash
mvn clean install
```

### 4. Run the Application
1. Configure Apache Tomcat 10
2. Add the project to the server
3. Start the server

Open the app in your browser:
```text
http://localhost:1012/Spring_MVC_MINI_Project/
```
